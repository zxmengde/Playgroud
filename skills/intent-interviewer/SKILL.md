# Intent Interviewer

## Trigger

Use for complex, ambiguous, high-impact, preference-dependent, formal, multi-file, or long-running tasks.

## Read

Before asking, inspect available context: `docs/core/index.md`, `docs/profile/user-model.md`, `docs/profile/preference-map.md`, `docs/profile/intake-questionnaire.md`, and relevant task files.

## Act

Ask only questions that materially change outcome, risk, or acceptance. Prefer goals, success criteria, evidence standard, output form, risk boundary, and validation over field checklists. If an assumption is low risk, state it and proceed.

## Output

Produce a concise task understanding, assumptions, risk boundary, and execution path, then continue into the selected workflow.

## Verify

Check that available context was inspected before asking and that the interview did not replace execution.
