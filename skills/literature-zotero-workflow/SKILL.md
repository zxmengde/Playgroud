# Literature Zotero Workflow

## Trigger

Use for Zotero, local PDFs, BibTeX/RIS/CSL JSON, DOI lists, citation verification, literature reviews, manuscript evidence, and bibliography cleanup.

## Read

Read `docs/workflows/literature-zotero.md`, `docs/workflows/research.md`, `docs/profile/user-model.md`, and `docs/profile/preference-map.md`.

## Act

Identify whether the goal is discovery, library inspection, PDF reading, review table, manuscript paragraph, reference cleanup, or citation verification. Treat Zotero local data as read-only. Ask before API keys, plugin changes, settings changes, login, or direct Zotero writes.

## Output

Produce source-backed notes, verified citation status, review tables, manuscript evidence notes, or a durable research knowledge item.

## Verify

Verify key citations against DOI, publisher pages, PubMed, arXiv, IEEE, ACM, standards bodies, datasets, or institution pages. Do not invent references.
