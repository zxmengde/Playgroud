# 系统概览

该系统用于把 Codex 组织为受控、可审计、可持续修正的个人智能工作系统，服务 Windows 环境下的科研、办公、编码、文档处理、网页访问、知识沉淀和任务跟进。

系统不依赖单一提示词。稳定能力由项目级规则、技能、工具登记、知识库、模板、任务记录、权限规则和复盘记录共同构成。

仓库内 `skills/` 保存本机 Codex 技能组的同步副本，使云端 Codex 克隆仓库后仍能读取同一套工作规则。

系统默认降低用户负担。用户只需要表达目标；Codex 负责同步仓库、读取上下文、判断任务复杂度、必要时进行自适应访谈、执行任务、验证结果，并在发现未来行为需要改进时主动提出确认。

## 适用范围

适用任务包括：

- 科研问题拆解、文献检索、论文与 PDF 阅读、资料摘要、引用线索整理。
- Word、PPT、Excel、PDF、Markdown、网页资料等办公与学习文件处理。
- 代码库理解、修改、测试、错误排查、文档生成和代码审查。
- 网页访问、资料提取、截图、下载前检查和来源留存。
- 长期偏好、项目背景、术语、模板和经验的本地沉淀。

## 能力边界

该系统不是无人值守的外部账号代理。外部账号授权、发送消息、提交表单、购买、发布、远程写入和长期运行服务需要明确确认。

Codex Memories 只作为辅助召回。稳定事实、偏好和流程应写入本仓库或 Codex 技能。

## 运行原则

每次工作应遵循：读取上下文、确认风险边界、执行可完成事项、验证结果、记录产物、沉淀可复用经验。
