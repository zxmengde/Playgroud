# 权限规则

## 可直接执行

以下操作可直接执行：

- 读取本地文件和项目结构。
- 检索公开资料并记录来源。
- 创建新文件、草稿、模板、知识条目和检查报告。
- 运行非破坏性命令、测试、构建、渲染和结构检查。
- 截图、网页读取、资料提取和只读浏览器操作。
- 整理任务记录、复盘记录和工具登记。

## 需确认执行

以下操作执行前应确认：

- 删除、覆盖、大规模移动或批量重命名文件。
- 向外部账号写入内容。
- 提交表单、发送消息、购买、发布或上传。
- 安装长期运行服务、计划任务或后台代理。
- 保存可能敏感的信息。
- 修改账号权限、密钥、令牌或凭据配置。

## 禁止执行

以下操作不得执行：

- 保存或复述密钥、令牌、账号密码。
- 绕过系统、网站或账号的权限限制。
- 未经确认执行不可逆的重要资料修改。
- 将敏感资料写入公开位置。

## 记录要求

复杂任务应记录关键输入、读取来源、产物路径、验证方式和剩余风险。阻塞任务写入 `docs/tasks/blocked.md`。
